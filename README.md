# RAMS4 GUI

---

## Setup & Running Locally

### Prerequisite

- [Node.js](https://nodejs.org/en) (v20+)

### Install & Run

```bash
git clone https://github.com/khakis23/RAMS4-GUI.git
cd RAMS4-GUI
npm install
npm run dev
```

Navigate to the `localhost` link provided in the terminal.
